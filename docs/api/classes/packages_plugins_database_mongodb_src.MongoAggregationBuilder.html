Class MongoAggregationBuilder<T>

Fluent builder for MongoDB aggregation pipelines

Example

const pipeline = new MongoAggregationBuilder()
  .match({ status: 'active' })
  .group({ _id: '$country', count: { $sum: 1 } })
  .sort({ count: -1 })
  .limit(10)
  .build();
